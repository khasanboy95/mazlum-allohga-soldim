# Google'da chiqishi uchun qo'llanma

Bu papkadagi fayllar qo'shiq uchun tayyor SEO sahifa:

- `index.html`
- `mazlum-allohga-soldim.mp3`
- `robots.txt`
- `sitemap.xml`

## Muhim

Google natijalarida chiqishni hech kim darhol majburlab bera olmaydi. Qo'shiq Google'da ko'rinishi uchun fayllar internetda ochiq URL orqali turishi, Google sahifani indekslashi va qidiruv natijalariga qo'shishi kerak.

## Nima qilish kerak

1. Domen yoki hosting oling. Masalan: Netlify, Vercel, GitHub Pages yoki oddiy hosting.
2. `YOUR-DOMAIN.example` matnini o'z domeningizga almashtiring.
3. Fayllarni hostingga yuklang. Tavsiya qilingan URL:
   `https://sizning-domeningiz.uz/mazlum-allohga-soldim/`
4. Google Search Console'ga kiring: https://search.google.com/search-console
5. Domeningizni qo'shing va tasdiqlang.
6. `sitemap.xml` yuboring.
7. URL Inspection orqali sahifani indekslashga yuboring.

## Qidiruvda yaxshi chiqishi uchun

- YouTube, Spotify, Apple Music, SoundCloud, Telegram kanal va Instagram postlarida shu sahifaga link bering.
- Sahifa nomida aynan `Mazlum - Allohga soldim` yozilgan bo'lsin.
- Boshqa saytlar ham shu sahifaga link bersa, Google tezroq va yaxshiroq topadi.
- Mualliflik huquqi sizda yoki ruxsatingiz borligiga ishonch hosil qiling.
